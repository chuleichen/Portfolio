Source File

Of possible interests, you can download the world and explore more by yourself.

Here is the source file and required configuration.

  • Minecraft version: Java edition 1.12.2
  • Resourse pack: Cocricot
  • Shader pack: Sildur Vibrant Shaders V1.20 Extreme
  • Required mod for exploration:
    1. Cocricot 1.10.3
    2. Conquest Reforged 3.0.2
    3. OptiFine 1.12.2
    4. Not enough ids 1.5.4.4
  • Required mod for further build:
    1. Liteloader 1.12.2
    2. Worldedit forge 6.1.10
    3. Worldedit CUI

Warning

Please do not challenge your CPU and GPU if you are using a laptop or an old computer. Try to turn off the shaders or set a smaller loading area.
